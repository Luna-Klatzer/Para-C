*************
Pre-Processor
*************

.. toctree::
    :hidden:

This serves as a base reference for the Para-C Pre-Processor, which is per
default integrated in the Para-C Compiler. This Pre-Processor is designed after
the `GCC C Pre-Processor <https://gcc.gnu.org/onlinedocs/cpp/>`_ and extends it
for the usage in Para-C and its features.
