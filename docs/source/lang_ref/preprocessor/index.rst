*************
Pre-Processor
*************

.. toctree::
    :hidden:

This serves as a base reference for the Para Pre-Processor, which is per
default integrated in the Para Compiler. This Pre-Processor is designed after
the `GCC C Pre-Processor <https://gcc.gnu.org/onlinedocs/cpp/>`_ and extends it
for the usage in Para and its features.
