Using the Command Line Interface
================================

Available commands
------------------

+------------------------+--------------------------------------------------------------------------------------------------+
| Name                   | Description                                                                                      |
+========================+==================================================================================================+
| ``para compile``       | Compiles a Para program to C or an executable.                                                   |
+------------------------+--------------------------------------------------------------------------------------------------+
| ``para run``           | Compiles a Para program and runs it.                                                             |
+------------------------+--------------------------------------------------------------------------------------------------+
| ``para analyse``       | Analyses a program and validates the syntax                                                      |
+------------------------+--------------------------------------------------------------------------------------------------+
